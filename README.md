# maxcoga.github.io
 personal website

created using react.

**Init**
- npm init react-app max-app

**Deploy**
- cd max-app
- npm run deploy

**Update**
- npm install react-scripts@latest

**dev-dependency**
- cd max-app
- npm install gh-pages --save-dev

**package properties**
- add "homepage": "https://maxcoga.github.io"
